Colleagues often rely on me when systems become difficult to maintain, slow, or unstable. I enjoy investigating root causes, improving existing systems, and designing solutions that simplify complex workflows.

My strengths include deep problem investigation, system-level thinking, careful documentation, and building solutions that automate repetitive work and save time and resources.

Currently interested in projects involving system architecture, automation, and emerging technologies such as AI-assisted workflows and tools like n8n. I enjoy environments where thoughtful engineering, clear requirements, and well-structured collaboration allow complex problems to be solved properly.
